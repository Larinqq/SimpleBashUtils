Simple Bash Utils
Реализация утилит командной строки cat и grep на языке C.

О проекте
Проект представляет собой реализацию двух классических Unix-утилит для работы с текстовыми файлами:

s21_cat — утилита для отображения, объединения и создания текстовых файлов

s21_grep — утилита для поиска текста с использованием регулярных выражений

Обе программы разработаны на языке C стандарта C11, следуют принципам структурного программирования и стилю кодирования Google Style. Сборка осуществляется с помощью Makefile.

Технологии
Язык: C (стандарт C11)

Компилятор: gcc

Сборка: Makefile

Регулярные выражения: библиотеки PCRE или regex

Тестирование: интеграционные тесты с сравнением поведения с оригинальными утилитами Bash

Установка и сборка
bash
# Клонирование репозитория
git clone <repository-url>
cd simple-bash-utils

# Сборка cat
make s21_cat

# Сборка grep
make s21_grep

# Очистка объектных файлов
make clean
Использование
s21_cat
Утилита для работы с текстовыми файлами. Поддерживает следующие опции:

Опция	Описание
-b	нумерует только непустые строки
-e	отображает символы конца строки как $
-n	нумерует все выходные строки
-s	сжимает несколько смежных пустых строк
-t	отображает табуляции как ^I
Примеры использования:

bash
# Вывод содержимого файла с нумерацией строк
./s21_cat -n file.txt

# Вывод с отображением символов конца строк
./s21_cat -e file.txt

# Сжатие пустых строк
./s21_cat -s file.txt
s21_grep
Утилита для поиска текста с использованием регулярных выражений. Поддерживаемые опции:

Опция	Описание
-e	задает шаблон поиска
-i	игнорирует регистр символов
-v	инвертирует поиск (выводит строки, не соответствующие шаблону)
-c	выводит только количество совпадающих строк
-l	выводит только имена совпадающих файлов
-n	выводит номера строк
-h	выводит совпадающие строки без имен файлов
-s	подавляет сообщения об ошибках

Примеры использования:

bash
# Поиск слова в файле
./s21_grep "pattern" file.txt

# Поиск с игнорированием регистра и выводом номеров строк
./s21_grep -i -n "pattern" file.txt

# Подсчет количества совпадений
./s21_grep -c "pattern" file.txt

# Инвертированный поиск (вывод строк, НЕ содержащих pattern)
./s21_grep -v "pattern" file.txt

# Поддержка комбинаций флагов
./s21_grep -iv "pattern" file.txt

Различные комбинации флагов

Различные входные данные
